# EasiUserTools

**EasiUserTools** 是一个用于批量添加自定义 **希沃白板** 用户的工具。  
它可以通过图片文件快速创建用户。

---

## 使用方法

- 向`./targets`中添加图片，文件名为期望的用户名
- 运行`main.py`
- 效果如下
 ![image](/assets/preview.png "效果预览")
---

## 安装

1. 克隆仓库：

```bash
git clone https://github.com/yourusername/EasiUserTools.git
cd EasiUserTools
uv sync
```
